Clients e Servers

Quando falamos de internet, precisamos entender o que são Clients e Servers. Clients é o lado do dispositivo que está acessando aquele congeúdo, como computadores domésticos, smartphones, tablets, notebooks e até mesmo relógios e automóveis inteligentes nos dias de hoje. Servers é onde fica hospedado todo o conteúdo acessado. Os servidores podem (ou não) ter acesso à banco de dados para acessar cadastros ou buscar conteúdos em nuvem.

Aplicações Web

As aplicações Web são conteúdos ou soluções que possuem internet como meio de comunicação entre client e server sem ser necessária a instalação de uma aplicação. Isso acontece, por exemplo, ao acessar o Facebook por um navegador, ou o YouTube. Você está consumindo um conteudo visualmente similar ao do aplicativo, mas completamente hospedado online.

"HTML NÃO É UMA LINGUAGEM DE PROGRAMAÇÃO, mas sim um arquivo de demarcação interpretado pelos navegadores."

Principais Tags e Suas Funções

Diversas funções ajudam a personalizar e formatar sua página HTML. Entre elas temos as principais:

  • a: Serve para adicionar um link a página;
  • hr: Adiciona uma linha na formatação;
  • h1 / h2 / h3 / etc: Cria uma sequência de subtítulos;
  • p: Cria um novo parágrafo de texto;
  • ol / ul: Cria lista ordenada e não ordenada (respectivamente);
  • li: Cria uma linha na lista em questão;
  • strong / i / u: Formata o texto em negrito, itálico e sublinhado (respectivamente);
